# Arduino Power Outage Protection System

This project simulates a system that protects electrical appliances
from frequent power outages using Arduino UNO.

## Features
- Detects frequent power outages
- Uses RTC to track outage timing
- Cuts power using a relay if outages exceed limits
- Automatic restore after delay

## Components
- Arduino UNO
- RTC DS1307
- Relay Module
- Buzzer
- Push Button
